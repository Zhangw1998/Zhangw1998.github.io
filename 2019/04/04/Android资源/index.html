Android资源

2019-04-04Android

资源种类

  • values目录
    • 字符串资源（strings.xml）
    • 颜色资源 （colors.xml）
    • 尺寸资源 （dimens.xml）
    • 数组资源 （arrays.xml）
    • 样式资源和主题（styles.xml）
  • Drawable资源（drawable目录）
  • 菜单资源（menu目录）
  • XML资源（xml目录）
  • assets资源（assets目录）
  • 启动器图标资源（mipmap目录）

一、values目录

1、strings.xml

  1. 文件内容
    <resources>
 <string name="app_name">App Name</string>
</resources>
  2. 基本使用

    在xml中引用： @string/app_name

    在活动中使用：String appName = getResources().getString(R.string.app_name);

  3. 国际化

    1.右键点击res目录 -> New -> Android Resource Directiory

    2.在弹出框中下方左侧列表中选择Locale，然后点击中间的“右移按钮”（>>）

    3.在Language列表和Specific Region Only列表选择对应的选项，最后点击OK按钮，就会在res目录下生成对应的values目录

    参考: Android Studio创建对应国家语言values

2、colors.xml

  1. 文件内容
    <?xml version="1.0" encoding="utf-8"?>
<resources>
 <color name="colorPrimary">#008577</color>
 <color name="colorPrimaryDark">#00574B</color>
 <color name="colorAccent">#D81B60</color>
</resources>
  2. 基本使用

    在xml文件中使用： @color/colorPrimary

3、demens.xml

  1. 文件内容
    <?xml version="1.0" encoding="utf-8"?>
<resources>
<dimen name="activity_margin">16dp</dimen>
</resources>
  2. 基本使用

    在xml文件中使用： @dimen/activity_margin

  3. 适配屏幕

    参考一种非常好用的Android屏幕适配

4、arrays.xml

  1. 文件内容
    <?xml version="1.0" encoding="utf-8"?>
<resources>
 <string-array name="string_array">
     <item>1</item>
     <item>2</item>
     <item>3</item>
 </string-array>
</resources>
  2. 基本使用

    在ListView中使用：android:entries=”@array/string_array”

    在活动中使用：String[] array = getResources().getIntArray(R.array.string_array);

4、styles.xml

  1. 文件内容

    <resources>
 <!-- Base application theme. -->
 <style name="AppTheme" parent="Theme.AppCompat.Light.DarkActionBar">
     <!-- Customize your theme here. -->
     <item name="colorPrimary">@color/colorPrimary</item>
     <item name="colorPrimaryDark">@color/colorPrimaryDark</item>
     <item name="colorAccent">@color/colorAccent</item>
 </style>

 <style name="TextStyle">
     <item name="android:layout_width">60dp</item>
     <item name="android:layout_height">30dp</item>
     <item name="android:textColor">#000000</item>
 </style>
</resources>
  2. 基本使用

    主题是在活动中使用，即在Manifest.xml文件中配置（theme标签）
    样式是在控件中使用，在控件的style标签中引用（@style/TextStyle）

二、Drawable资源

  1. 文件类型

    xml: 可用来编写控件的背景，比如给EditText加上边框（shape）

    png: 图片类型，src/background 标签中引用，不同分辨率的图片应放在不同的drawable目录下

  2. 基本使用

    drawable/filename

  3. shape
    <?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android"
 android:shape="rectangle">
 <padding
     android:bottom="7dp"
     android:left="7dp"
     android:right="7dp"
     android:top="7dp" />
 <stroke
     android:width="5px"
     android:color="#000" />
 <solid android:color="#FFFFFF"/>
</shape>

    参考: Android XML shape 标签使用详解

三、菜单资源

  1. 文件内容
    <?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android">
<item
 android:id="@+id/menu_settings"
 android:title="设置"/>
</menu>
    2.基本使用

    在活动中使用：

    重写onCreateOptionsMenu()方法，用于在活动中显示菜单

    @Override
public boolean onCreateOptionsMenu(Menu menu) {
 getMenuInflater().inflate(R.menu.main, menu);
 return true;
}

    重写onOptionsItemSelected()方法，用于处理点击菜单项的事件

    @Override
public boolean onOptionsItemSelected(MenuItem item) {
 switch (item.getItemId()) {
     case R.id.menu_settings:
         // 具体的响应事件
         break；
 }
 return super.onOptionsItemSelected(item);
}

四、XML资源

一般作为配置文件，如果需要读取相应的数据，需要使用XML解析。

五、assets资源

  1. assets目录下的资源不会被编译
  2. 在活动中使用：
     InputStream is = getAssets().open("filename");
 byte[] bytes = new byte[is.available()];  // 当文件过大时应使用循环来读取数据

六、mipmap资源

mipmap目录的图标会忽略屏幕密度，回去匹配大一点的，然后系统自动对图片进行缩放，从而优化显示和节省资源。官方建议如下类型的图片资源放在mipmap目录：

  • launcher icons
  • action bar and tab icons
  • Notification ico
密度 建议尺寸
mipmap-mdpi 48 * 48
mipmap-hdpi 72 * 72
mipmap-xhdpi 96 * 96
mipmap-xxhdpi 144 * 144
mipmap-xxxhdpi 192 * 192
